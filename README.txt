Pasos a seguir para la debida instalación y funcionamiento del sistema

1._ Importar todo el proyecto Git a un entorno Apache de preferencia personal Xampp.
2._ abrir el archivo con nombre "bd.sql", copiar y pegar el contenido en la sección de "SQL" en Mysql o directamente "Importar" el archivo, para crear la base de datos y tablas necesarias (por defecto viene con 3 productos)
3._ Puede ingresar al index y ver el funcionamiento de dicho sistema; con los botones de acción y consultas de preferencias.
